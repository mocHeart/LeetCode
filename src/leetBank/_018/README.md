# [ 四数之和 ][title]

## 描述
给定一个包含 n 个整数的数组 nums 和一个目标值 target，
判断 nums 中是否存在四个元素 a，b，c 和 d ，使得 a + b + c + d 的值与 target 相等？
找出所有满足条件且不重复的四元组。

注意：
答案中不可以包含重复的四元组。

**示例:**
```
给定数组 nums = [1, 0, -1, 0, -2, 2]，和 target = 0。

满足要求的四元组集合为：
[
  [-1,  0, 0, 1],
  [-2, -1, 1, 2],
  [-2,  0, 0, 2]
]
```


**标签:** 数组 &  双指针


## 思路
解法： 定2双指指针
    和之前三数之和类似，固定两个数，再将两个指针分别指向剩余数字的首尾，具体步骤类似

## 结语
  勤加练习，注意细节。
