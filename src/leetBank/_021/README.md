# [ 合并两个有序链表 ][title]

## 描述
将两个升序链表合并为一个新的升序链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。 

**示例:**
```
输入：1->2->4, 1->3->4
输出：1->1->2->3->4->4
```


**标签:** 链表


## 思路
解法1： 迭代
   创建一个dummy头结点，迭代两列表依次向头结点连接较小的节点
   
解法2：递归
   以其中一个链表为中心，递归整合另一个链表

    
## 结语
    熟悉递归的编程方式。
