# [ 在排序数组中查找元素的第一个和最后一个位置 ][title]

## 描述
给定一个按照升序排列的整数数组 nums，和一个目标值 target。
找出给定目标值在数组中的开始位置和结束位置。


**示例:**
```
示例 1:
输入: nums = [5,7,7,8,8,10], target = 8
输出: [3,4]

示例 2:
输入: nums = [5,7,7,8,8,10], target = 6
输出: [-1,-1]

```

**标签:**  数组 & 二分查找


## 思路
解法1: 二分查找到某个目标位置，在左右扩展至边缘
解法2: 基于查找左右边界的二分法

## 结语
  查找左右边界的二分法是一个良好的拓展，需熟记！
