# [ 两两交换链表中的节点 ][title]

## 描述
给定一个链表，两两交换其中相邻的节点，并返回交换后的链表。
你不能只是单纯的改变节点内部的值，而是需要实际的进行节点交换。

**示例:**
```
给定 1->2->3->4, 你应该返回 2->1->4->3.
```


**标签:**  链表 


## 思路
解法1：递归
    剩余至少两个节点，第一个节点的下一个为递归返回值（下两个的头结点），第二个节点的下一个节点为第一个节点 
    时间复杂度:O(N)  空间复杂度:O(N)
    
解法2： 迭代
    添加前置dummy节点帮助处理，同时每次迭代的指针的指向变化
    时间复杂度:O(N)  空间复杂度:O(1)
    
## 结语
  循环问题，注意理解怎么构成的循环，每次循环变量的变化。
