# [ 括号生成 ][title]

## 描述
数字 n 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且有效的括号组合。

**示例:**
```
输入：n = 3
输出：[
       "((()))",
       "(()())",
       "(())()",
       "()(())",
       "()()()"
     ]
```


**标签:** 字符串 & 回溯算法


## 思路
解法1： 递归回溯
    字符串长度 == 2n  完成一个项, 然够用left、right记录左、右括号的数目，递归添加

解法2： 括号长度递归
    回溯到n=1，在次基础上依次向上计算n=2,3...的所有可能
